# Basic_CSS-Assignment
This project demonstrates the use of an external CSS file to style an HTML page based on a provided visual example.

The goal of the assignment was to closely replicate the appearance of the reference screenshot by applying CSS for layout, fonts, colors, spacing, links, lists, and tables.

Features implemented:

Light blue page background to match the example layout

Styled headers with proper sizing and alignment

A highlighted second paragraph using custom width, padding, border, and line height

Dark blue formatted paragraphs for emphasis

Two-digit ordered list formatting

Different link colors for list items and the W3Schools reference link

A styled table with separate header and cell backgrounds

Collapsed table borders applied to all cells
